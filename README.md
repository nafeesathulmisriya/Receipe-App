🍲 Recipe Management System

The Recipe Management System is a web application developed using Python and Django that allows users to browse recipes and administrators to manage recipes efficiently.
The system provides features such as searching recipes, filtering recipes by category, and full recipe management (add, view, edit, delete) through the admin interface.

📌 Features
👨‍🍳 Recipe Management

Add new recipes
View recipe details
Edit existing recipes
Delete recipes

🔍 Search Functionality
Users can search recipes by name to quickly find a specific recipe.

🗂 Category-wise Recipes
Recipes are organized into categories for easy browsing.

📖 Recipe Details
Each recipe displays:
Recipe title
Description
Ingredients
Preparation time
Recipe image

🏠 Home Page
Displays all available recipes in a user-friendly layout.

⚙ Admin Panel
Admin can manage all recipes and categories from the Django admin panel.

🛠 Technologies Used
Backend: Python
Framework: Django
Frontend: HTML, CSS, Bootstrap
Database: SQLite
Version Control: Git & GitHub
